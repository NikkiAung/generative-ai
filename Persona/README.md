Persona-Based Prompting Assignment
Objective
Understand how changing the AI persona affects tone, explanation style, and output quality.
Part 1 - Basic Prompt (No Persona)
• Use the prompt: 'Explain what an API is.'
• Record the response.
• Identify the tone (formal / casual / technical).
• Note the length of explanation.
Part 2 - Add Different Personas
• Teacher Persona: You are a friendly computer science teacher explaining concepts to
beginners. Explain what an API is.
• Senior Backend Engineer Persona: You are a senior backend engineer with 10 years of
experience. Explain what an API is.
• 10-Year-Old Child Persona: You are explaining to a 10-year-old child. Explain what an API is.
Part 3 - Compare Outputs
• How did the tone change?
• Did the complexity level change?
• Which explanation was most clear?
• Which persona would you use in a classroom, client meeting, or technical blog?
Part 4 - Practical Application
• Explain what Docker is using:
• No persona
• Teacher persona
• DevOps engineer persona
• Write 5-6 lines comparing the outputs.
Submission Format
• All prompts used
• All outputs
• Short comparison summary (8-10 lines)
